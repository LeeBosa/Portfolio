웹 프론트엔드 개발 / 퍼블리싱

프론트엔드 웹 개발과 웹 퍼블리싱에 필요한 언어들

HTML

(Hyper Text Markup Language)
각각의 요소(Elements)와 태그를 이용해 웹페이지의 기본 구조를 구성하는데 능숙합니다.

CSS

(Cascading Style Sheet)
웹의 구성 요소들을 디자인된 레이아웃에 맞춰서 스타일을 입히는데 능숙합니다.

SCSS

(Sassy Cascading Style Sheet)
CSS에서 구현하기 힘든 복잡한 요소들을 SCSS를 이용해 구현하는데 능숙합니다.

JavaScript

(Ecma Script Language)
웹의 동적인 요소들을 제어하고, 데이터 연동에 따른 웹의 변화를 제어하는데 능숙합니다.

JQuery

(Javascript Library)
풍부한 라이브러리, 그리고 AJAX, JSON, XML을 통한 데이터 파싱을 다루는데 능숙합니다.

React

(JavaScript Library)
모듈과 Virtual DOM을 활용하여 HTML DOM 요소를 제어하는데 능숙합니다.

디자인 프로그램

웹 디자인과 제품 디자인에 필요한 기술들

PhotoShop

(Adobe Design Program)
배경 제거, 컬러톤 보정 등 이미지 작업에 관련된 전반적인 작업에 능숙합니다.

Illustrator

(Adobe Design Program)
SVG 파일 제작, 일러스트 작업 등 일러스트레이터에 관련된 작업에 능숙합니다.

InDesign

(Adobe Design Program)
출판 작업에 관한 전반적인 이해도가 높고, 인쇄물 편집에 능숙합니다.

Figma

(Cloud-base Design Program)
피그마를 이용해 디자인팀과의 협업, 디자인 작업에 능숙합니다.
