프론트엔드 개발

프론트엔드 개발에 필요한 언어, 프로그램

HTML

(Hyper Text Markup Language)
적절한 태그를 활용해 웹페이지의 기본 구조를 작성하는데 능숙합니다.

CSS

(Cascading Style Sheet)
웹의 구성 요소들을 디자인된 레이아웃에 맞춰서 스타일을 입히는데 능숙합니다.

SASS, SCSS

(Sassy Cascading Style Sheet)
CSS에서 구현하기 힘든 복잡한 요소들을 SCSS를 사용해 구현하는데 능숙합니다.

Markdown

(Markup Language)
마크다운 언어를 사용해 간단한 HTML 구조의 문서 작성에 능숙합니다.

JavaScript

(Ecma Script Language)
웹의 동적인 요소들을 제어하고, 데이터 연동에 따른 웹의 변화를 제어하는데 능숙합니다.

JQuery

(Javascript Library)
다양한 라이브러리를 통해 동적인 요소를 구현하고 제어하는데 능숙합니다.

GSAP

(Javascript Library)
GSAP을 사용해 스크롤 트리거, 스크롤 트위닝 등 다양한 스크롤 효과를 구현하는데 능숙합니다.

Three.js

(Javascript Library)
Three.js를 사용해 다양한 3D Object를 구현하고 제어하는데 능숙합니다.

Node.js

(Javascript Runtime)
Node.js를 사용해 웹서버를 빌드하고 패키지를 사용해 다양한 효과를 구현하는데 능숙합니다.

React.js

(JavaScript Library)
컴포넌트와 props를 사용해서 HTML 요소를 구현하는데 능숙합니다.

AJAX

(Async JavaScript And XML)
AJAX를 통해 비동기로 들어오는 데이터를 제어하고 파싱하는데 능숙합니다.

JSON

(Javascript Object Notation)
JSON 형태로 들어오는 데이터를 가공해 정렬하고, 파싱하는데 능숙합니다.

Jekyll

(Static Site Generator)
Jekyll을 사용해 다양한 블로그의 Theme을 원격으로 구현하는데 능숙합니다.

Git, GitHub

(Version Control System, Community)
Git의 브랜치를 활용해 버전을 업데이트하고 협업하는데 능숙합니다.

SourceTree

(Version Control System)
SourceTree의 브랜치를 활용해 버전을 업데이트하고 협업하는데 능숙합니다.

제품 디자인

제품 디자인에 필요한 프로그램

PhotoShop

(Adobe Design Program)
배경 제거, 컬러톤 보정 등 이미지 작업에 관련된 전반적인 작업에 능숙합니다.

Illustrator

(Adobe Design Program)
SVG 파일 제작, 일러스트 작업 등 일러스트레이터에 관련된 작업에 능숙합니다.

Figma

(Cloud-base Design Program)
피그마를 이용해 협업, 간단한 일러스트 작업 및 포토 에디팅에 능숙합니다.

Rhinoceros

(3D Rendering Program)
Rhinoceros는 3D 프로토타입 제작에 적합한 3D 디자인 프로그램입니다.

SolidWorks

(3D Rendering Program)
3D 제품에 대한 디테일한 공정을 설계할 때 사용하는 프로그램입니다.

KeyShot

(3D Rendering Program)
제작한 3D 파일에 컬러, 텍스쳐를 입히고, 카메라 구도, 밝기 등을 조정해 렌더링하는데 능숙합니다.
